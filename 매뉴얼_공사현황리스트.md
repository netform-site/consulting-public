### 2-2. 상단 버튼
| 버튼 | 기능 |

### 2-3. 통계 카드
| 카드 | 내용 |
|------|------|
| ✅ 입금완료 | 잔금까지 모두 입금된 현장 수 및 금액 |
| 📝 계약금 입금 | 계약금만 입금된 현장 수 및 금액 |
| 🔄 중도금 진행 | 중도금 입금 진행 중인 현장 수 및 금액 |

- **입금상태 필터**: 전체 / 입금완료 / 계약금입금 / 중도금진행 / 미입금 / 진행중 / 해지
- **공사유형 필터**: 전체 / 직영(본) / 직영(외) / 컨설팅(본) / 컨설팅(외)

## 3. 공사현황 관리

### 3-1. 공사 목록 보기
메인 테이블에 다음 정보가 표시됩니다:
- NO, 지역, 공사현장, 공사명
- 계약금액 (VAT별도 / VAT포함)
- 계약일, 착공일, 준공일
- 수금상태 (계약금/중도금/잔금 입금 여부)
- 유형, 담당자, 현장소장

### 3-2. 새 공사 추가
1. 우측 상단의 **+ 새 공사 추가** 버튼 클릭
2. 기본정보 입력:
   - NO, 지역, 공사유형, 공사현장, 공사명
   - 영업담당자, 현장소장
3. 금액 입력:
   - 계약금액(VAT별도) 입력 → VAT포함 금액 자동 계산
4. 날짜 입력:
   - 계약일, 착공일, 준공일
5. 수금내역 입력 (선택):
   - 계약금 / 중도금 / 중도금2 / 잔금
   - 각각 금액, 계산서 발행일, 입금일 입력
6. **저장** 클릭

### 3-3. 공사 수정
1. 목록에서 해당 현장의 **수정** 버튼(✏️) 클릭
2. 수정할 내용 변경
3. **저장** 클릭

### 3-4. 공사 삭제
- **단건 삭제**: 해당 현장의 삭제 버튼(🗑️) 클릭 → 확인
- **일괄 삭제**: 체크박스로 여러 건 선택 → **선택 삭제** 버튼 클릭 → 확인

### 3-5. 엑셀 업로드
1. **엑셀 업로드** 버튼 클릭 (또는 파일을 드래그&드롭)
2. .xlsx, .xls, .csv 파일 선택
3. 기존 엑셀 양식의 열 순서에 맞게 자동으로 데이터 등록
4. 업로드 결과 확인

### 3-6. 엑셀 다운로드
1. **다운로드** 버튼 클릭
2. 현재 년도의 전체 공사 데이터가 엑셀 파일로 저장됨

## 4. 현장 상세 (총괄표)

### 4-1. 상세 열기
- 공사 목록에서 해당 현장의 **상세** 버튼 클릭

### 4-2. 총괄표 탭

#### 기본정보
- 공사현장, 공사명, 공사기간, 공사금액

#### 수금내역 (왼쪽)
| 구분 | 입력항목 |
|------|----------|
| 계약금 | 일자, 금액, 내용 |
| 중도금1차 | 일자, 금액, 내용 |
| 중도금2차 | 일자, 금액, 내용 |
| 잔금 | 일자, 금액, 내용 |
| 기타입금1~2 | 일자, 금액, 내용 |
| 금액조정 | 일자, 금액, 내용 |

→ 하단에 합계가 자동 계산됩니다.

#### 투입내역 (오른쪽)
- **+ 자재비** 버튼: 자재비 항목 추가
- **+ 노무비** 버튼: 노무비 항목 추가
- **+ 기타경비** 버튼: 기타경비 항목 추가
- 각 항목: 구분, 일자, 금액, 내용, 지급일
- 우측 🗑️ 버튼으로 개별 삭제

#### 상단 요약
- 계약금액, 자재비, 노무비, 기타경비, 지출계, 잔액이 자동 계산

#### 비고 / 사진 첨부
- 메모 입력 가능
- 사진 파일 첨부 가능 (복수 선택)

### 4-3. 정산서 탭
- 공사내역, 자재비, 노무비, 기타경비 세부 입력
- 총 매출액, 총 매입액, 손익, 손익률 자동 계산
- 현장소장 송부 기능

### 4-4. 총괄표 엑셀
- **📂 총괄표 불러오기**: 기존 엑셀 파일에서 데이터 가져오기
- **📥 총괄표 다운로드**: 입력된 내용을 엑셀로 저장
- **저장** 버튼: 입력 내용 클라우드 저장

### 4-5. 체크리스트
- 공사 목록에서 **체크** 버튼 클릭
- 18개 항목 (공사범위, 실착공일, 계약서, 준공검사 등) 진행 체크
- 완료율 자동 표시

## 5. 기술컨설팅 관리

### 5-1. 컨설팅 탭 전환
- 상단의 **📋 기술컨설팅** 탭 클릭

### 5-2. 통계 카드
- 총 현장 수, 컨설팅(Y) 건수, 총 낙찰금액, 총 실행금액

### 5-3. 신규 등록
1. **➕ 신규 등록** 버튼 클릭
2. 입력 항목:
   - 년도, 구분(일반/시지원/사후정산), 공사현장, 공사명
   - 낙찰금액, 담당자, 실행금액
   - 회사경비, 기타경비
3. 하단에 지출총액, 이익금, 이익률, 정산금액(15%) 자동 미리보기
4. **저장** 클릭

### 5-4. 검색 및 필터
- 현장명 검색
- 년도 필터: 2024 / 2025 / 2026
- 구분 필터: 일반 / 시지원 / 사후정산
- 담당자 필터

### 5-5. 수정 / 삭제
- 각 항목의 **수정** 버튼(✏️) / **삭제** 버튼(🗑️)

### 5-6. 엑셀 다운로드
- **📥 엑셀** 버튼 클릭하여 컨설팅 데이터 다운로드

## 6. 관리자 기능

### 6-1. 관리자 로그인
1. **🔒 관리자 전용** 버튼 클릭
2. 비밀번호 입력
3. 관리자 기능 활성화

### 6-2. 담당자별 실적 현황
- **👥 담당자별 실적 현황** 클릭
- 본사/외주별 담당자 실적 표시
- 계약건수, 공사금액(VAT별도/포함) 확인

### 6-3. 정산관리
1. **💰 정산 관리** 클릭
2. 전체 현장의 정산 현황 조회:
   - 공사금액, 지출총액, 이익금, 정산금액(15%), 회사수익(85%)
3. 검색/필터:
   - 현장명 검색, 년도, 구분(본사/외주), 담당자, 입금상태, 정산여부
4. 정산상태 변경:
   - 각 현장의 정산 드롭다운에서 **미정산 / 정산완료 / 진행중** 선택

### 6-4. 일괄정산 실행
1. 정산관리 화면 하단의 **🔒 일괄정산 실행** 클릭
2. **Step 1**: 정산유형(공사/컨설팅) 선택 → 기간/담당자 선택 → **다음** 클릭
3. **Step 2**: 담당자별 현장 확인 → 체크박스로 선택/해제 → **정산 실행** 클릭
4. 다음 담당자로 이동하며 반복
5. **Step 3**: 전체 결과 확인 → PDF 생성 / 이메일 발송

### 6-5. 대표님 보고서
1. 정산관리 하단의 **📋 본사 보고서** 또는 **📋 외주 보고서** 클릭
2. 년도, 정산상태, 담당자 필터 적용
3. 보고서 내용:
   - 요약: 총 현장 수, 총 공사금액, 이익금, 정산금액(15%), 회사수익(85%)
   - 담당자별 실적 테이블
   - 현장별 상세 내역
4. **🖨️ 인쇄/PDF** 또는 **📥 엑셀 다운로드** 가능

### 6-6. 담당자 비밀번호 관리
- **🔑 담당자 비밀번호 관리** 클릭
- 담당자별 비밀번호를 초기값(0000)으로 리셋

### 6-7. 관리자 비밀번호 변경
- **⚙️ 비밀번호 변경** 클릭
- 현재 비밀번호 확인 후 새 비밀번호 설정

## 7. 내 정산 확인 (담당자)

### 7-1. 로그인
1. **👤 내 정산 확인** 클릭
2. 담당자 이름 선택 + 비밀번호 입력
3. 로그인 버튼 클릭

### 7-2. 정산 현황 보기
- **🏗️ 공사정산**: 본인 담당 공사 현장의 정산 내역
- **📋 기술컨설팅**: 본인 담당 컨설팅 현장의 정산 내역
- **📊 통합**: 공사 + 컨설팅 통합 현황

### 7-3. 필터
- 년도 선택: 전체 / 2024 / 2025 / 2026 / 24년 미정산+25년
- 정산상태: 전체 / 정산완료 / 미정산 / 진행중

### 7-4. 비밀번호 변경
- **🔑 비밀번호 변경** 클릭
- 현재 비밀번호 확인 후 새 비밀번호 설정

### 7-5. 로그아웃
- **🚪 로그아웃** 클릭

## 8. 동시 수정 방지

### 8-1. 잠금 동작
- 누군가 현장을 **수정 모달**로 열면 자동으로 잠금이 걸립니다
- 다른 사용자가 같은 현장을 수정하려고 하면:
  > ⚠️ "OOO님이 수정 중입니다. 잠시 후 다시 시도해주세요."
- 수정 완료(저장) 또는 취소(닫기) 시 자동 잠금 해제

### 8-2. 적용 대상
- 공사 정보 수정 모달
- 현장 상세 (총괄표) 모달
- 기술컨설팅 수정 모달

### 8-3. 비정상 종료 대처
- 브라우저를 갑자기 닫거나 인터넷이 끊긴 경우
- **5분 후** 잠금이 자동으로 만료되어 다른 사용자가 수정 가능

## 9. 자주 묻는 질문

**Q: 다른 컴퓨터에서도 사용할 수 있나요?**
A: 네. URL로 접속하면 어디서든 동일한 데이터를 사용할 수 있습니다. 스마트폰에서도 가능합니다.

**Q: 데이터는 어디에 저장되나요?**
A: 클라우드(Firebase)에 자동 저장됩니다. 인터넷 연결이 필요합니다.

**Q: 두 사람이 동시에 수정하면 어떻게 되나요?**
A: 먼저 수정을 시작한 사람에게 잠금이 걸리고, 다른 사람은 수정 중 알림을 받습니다. 충돌이 자동으로 방지됩니다.

**Q: 엑셀 업로드 형식은 어떻게 되나요?**
A: 기존 엑셀 파일의 열 순서(NO, 지역, 유형, 현장명...)에 맞추면 자동 인식됩니다. .xlsx, .xls, .csv 파일을 지원합니다.

**Q: 담당자 초기 비밀번호는 무엇인가요?**
A: 0000 입니다. 로그인 후 비밀번호를 변경해주세요.

**Q: 정산(15%)은 어떻게 계산되나요?**
A: 이익금(공사금액 - 지출총액)의 15%가 담당자 정산금액이며, 나머지 85%가 회사수익입니다.

**Q: 데이터를 실수로 삭제했으면 어떻게 하나요?**
A: 삭제된 데이터는 복구가 어렵습니다. 삭제 전 항상 확인 메시지가 표시되니 주의해주세요.
