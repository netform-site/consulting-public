시작하기 전에

이 가이드가 하는 일: 공사현황리스트의 데이터를 Google Sheets에 자동으로 동기화합니다.

완성되면 이렇게 됩니다:
- 공사현황리스트에서 데이터 수정 → Google Sheets에 자동 반영
- Google Sheets에서 직접 수정 → 공사현황리스트 새로고침 시 반영
- Google Sheets에서 표/그래프 등 자유롭게 분석 가능

준비물:

  • Google 계정 (Gmail 계정이면 됩니다)
  • 소요 시간: 약 10~15분
1 Google Sheets 만들기
  1. 아래 링크를 클릭해서 Google Sheets를 엽니다:
    Google Sheets 열기 →
  2. 왼쪽 상단의 + (새 스프레드시트)를 클릭합니다.
    또는 "빈 스프레드시트"를 클릭합니다.
  3. 제목을 클릭해서 이름을 바꿉니다:
    공사현황 데이터 (아무 이름이나 괜찮습니다)
  4. 주소창의 URL에서 ID를 복사합니다.
    URL이 이렇게 생겼을 겁니다:
    https://docs.google.com/spreadsheets/d/여기가_스프레드시트_ID/edit
    /d/ 와 /edit 사이의 긴 문자열이 스프레드시트 ID입니다.
    예시: URL이 https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit 이면
    ID는 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms 입니다.
  5. 이 ID를 메모장이나 어딘가에 임시로 복사해 두세요. 다음 단계에서 사용합니다.

3 웹 앱으로 배포하기

  

    
코드를 인터넷에서 접근할 수 있는 "웹 앱"으로 배포합니다.

    
    
      
  1. 
        Apps Script 편집기 오른쪽 상단의 "배포" 버튼을 클릭합니다.
      
    2. 
      
  2. 
        "새 배포" 를 선택합니다.
      
    3. 
      
  3. 
        배포 유형 선택 화면에서:
        
      
          
    • 왼쪽의 ⚙ 아이콘 → "웹 앱" 을 선택합니다.
      • 
        
    
      
    4. 
      
  4. 
        설정을 입력합니다:
        
      
          
    • 설명: 공사현황 동기화 API (아무거나 OK)
      • 
          
    • 다음 사용자 인증정보로 실행: 나 (본인 계정)
      • 
          
    • 액세스 권한이 있는 사용자: 모든 사용자
      • 
        
    
        
    
          "모든 사용자"로 설정해야 합니다!

          (데이터 읽기/쓰기만 가능하며, Google 계정 접근은 아닙니다)

4 공사현황리스트에 연결하기

  

    
복사한 URL을 공사현황리스트에 입력합니다.

    
    
      
  1. 
        공사현황리스트 열기 →
      
    2. 
      
  2. 
        상단 헤더에서 ⚪ 아이콘 (흰색 원) 또는 🔄 Sheets 버튼을 클릭합니다.
      
    3. 
      
  3. 
        "Google Sheets 연동 설정" 팝업이 열립니다.
      
    4. 
      
  4. 
        "Apps Script 배포 URL" 입력란에 3단계에서 복사한 URL을 붙여넣기 합니다.
      
    5. 
      
  5. 
        "💾 URL 저장" 을 클릭합니다.
      
    6. 
      
  6. 
        "🔗 연결 테스트" 를 클릭하여 연결을 확인합니다.
        
    
          "✅ 연결 성공!" 이 나오면 완료!
        
    
        
    
          "❌ 연결 실패" 가 나오면?
    
          - URL을 정확히 복사했는지 확인
    
          - 배포 시 "모든 사용자" 접근으로 설정했는지 확인
    
          - 3단계를 다시 진행해보세요
        
    
      
    7.

자주 묻는 질문


  

    
Q. Google Sheets에서 수정하면 바로 반영되나요?

    
공사현황리스트 페이지를 새로고침해야 반영됩니다. 실시간은 아니지만, 새로고침할 때마다 최신 데이터를 가져옵니다.

  


  

    
Q. 인터넷이 안 될 때는 어떻게 되나요?

    
기존 Firebase + localStorage가 그대로 동작합니다. 인터넷이 복구되면 다음 저장/새로고침 시 자동 동기화됩니다.

  


  

    
Q. 동시에 두 곳에서 수정하면?

    
더 최근에 수정된 데이터가 우선됩니다 (연도 단위). 또한 기존 편집 잠금(edit-lock) 시스템이 동시 편집을 방지합니다.

  


  

    
Q. 코드를 수정하면 다시 배포해야 하나요?

    
Apps Script 코드를 수정하면 새 배포를 해야 합니다. 배포 → 배포 관리 → 연필 아이콘 → 버전 "새 버전" → 배포.

  


  

    
Q. Sheets URL을 잘못 입력했어요.

    
⚪ 아이콘이나 🔄 Sheets 버튼을 클릭해서 설정 화면을 열고, URL을 다시 입력하면 됩니다.

  


  

    
Q. 무료인가요?

    
네, 모두 무료입니다. Google Sheets, Apps Script 모두 Google 계정만 있으면 무료로 사용할 수 있습니다.
