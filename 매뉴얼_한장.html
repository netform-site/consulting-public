📋 1. 공사현황 관리
📅
년도 전환 - 상단 탭(2020~2026) 클릭하여 해당 년도 데이터 조회
🔍
검색/필터 - 현장명·공사명·담당자 검색 + 입금상태·유형 필터 조합
➕
새 공사 추가 - + 새 공사 추가 버튼 → 정보 입력 → 저장
✏️
수정 - 목록에서 ✏️ 아이콘 클릭 → 내용 수정 → 저장
🗑️
삭제 - 단건 삭제(🗑️) 또는 체크박스 선택 후 일괄 삭제
📤
엑셀 연동 - 업로드(.xlsx/.xls/.csv) / 다운로드(현재 년도 데이터)
📊
정렬 - 각 컬럼 헤더 클릭 시 오름차순/내림차순 정렬
🏗️ 2. 현장 상세 (총괄표)
📋 상세 클릭 → 💰 수금내역 → 📦 투입내역 → 🔄 자동계산 → 💾 저장
항목설명
💰 수금내역계약금 → 중도금(1~3차) → 잔금 (입금일·금액 입력)
📦 투입내역+자재비 +노무비 +기타경비 항목별 추가
📊 총괄표 엑셀기존 총괄표 불러오기(📂) / 다운로드(📥)
📷 사진·비고현장 사진 첨부 + 메모 기록
📄 정산서 탭매출·매입·손익 상세 작성 (자동 계산)
📝 3. 기술컨설팅 관리
📋 컨설팅 탭 → ➕ 신규 등록 → ✍️ 정보 입력 → 🔄 자동 계산 → 💾 저장
📝
입력 항목 - 년도, 구분(일반시지원사후정산), 현장명, 낙찰금액, 담당자
💰
자동 계산 - 이익금 · 정산금액(15%) 실시간 미리보기
🔍
필터 검색 - 년도(24/25/26) · 구분(일반/시지원/사후정산) · 담당자별
📊
통계 요약 - 하단에 전체 건수·금액·이익금·정산금 합계 표시
✅ 4. 공사 체크리스트
📋
사용법 - 목록에서 체크 버튼 클릭 → 체크리스트 모달 열기
✅
체크 항목 - 공사범위, 계약서, 착공계, 보험, 안전교육, 준공검사 등 18개 항목
📊
완료율 - 체크 진행률 자동 계산 → 목록에 퍼센트 표시
🔐 동시 수정 방지 시스템
상황시스템 동작
👤 A가 현장 수정 중Firebase에 자동 잠금 등록
👤 B가 같은 현장 수정 시도⚠️ "A님이 수정 중입니다" 알림 → 수정 차단
✅ A가 저장 또는 닫기잠금 즉시 해제 → B 수정 가능
⚠️ 비정상 종료 (브라우저 닫기)5분 후 잠금 자동 만료 → 수정 가능
💡 공사현황 수정, 현장 상세(총괄표), 기술컨설팅 수정 모두 적용됩니다.
🔒 5. 관리자 기능 (관리자 전용)
메인 화면에서 🔒 관리자 전용 클릭 → 비밀번호 입력 후 사용
기능설명
👥 실적 현황본사/외주별 담당자 계약건수·금액 통계 대시보드
💰 정산 관리전 현장 정산현황 조회·필터·상태 변경 (정산완료/미정산)
🔒 일괄정산공사/컨설팅 담당자별 선택 후 일괄 정산 실행
📋 보고서본사/외주 보고서 (PDF/엑셀 출력) 회사수익(85%) 포함
📥 엑셀 다운로드정산 데이터 전체 엑셀 내보내기
🔑 비밀번호 관리담당자 비밀번호 초기화(0000) / 관리자 비밀번호 변경
📌 일괄정산 3단계 프로세스
Step 1 유형·기간·담당자 선택 → Step 2 대상 현장 확인·실행 → Step 3 완료 확인·PDF·이메일
👤 6. 내 정산 확인 (담당자용)
👤 내 정산 확인 → 이름 선택 → 비밀번호 입력 → 로그인 완료
초기 비밀번호: 0000 (최초 로그인 후 반드시 변경 권장)
탭조회 내용
🏗️ 공사정산본인 담당 공사 현장별 이익금·정산금액(15%) 조회
📋 컨설팅정산본인 담당 컨설팅 현장별 이익금·정산금액(15%) 조회
📊 통합현황공사 + 컨설팅 합산 총 정산금액 현황
📌 필터 옵션: 년도(전체/2024/2025/2026) · 정산상태(전체/완료/미정산/진행중)

❓ 자주 묻는 질문 (FAQ)
Q: 다른 PC에서도 사용 가능한가요?
A: 네. URL 접속하면 어디서든 동일 데이터를 사용할 수 있습니다. (스마트폰·태블릿 가능)
Q: 데이터는 어디에 저장되나요?
A: 클라우드(Firebase)에 자동 저장됩니다. 인터넷 연결이 필요합니다.
Q: 담당자 초기 비밀번호는?
A: 0000입니다. 최초 로그인 후 반드시 변경해주세요.
Q: 엑셀 업로드 가능한 형식은?
A: .xlsx / .xls / .csv 파일을 지원합니다. 기존 양식 그대로 업로드하면 됩니다.
Q: 동시에 같은 현장을 수정하면 어떻게 되나요?
A: 먼저 수정하는 사람이 잠금을 획득하며, 다른 사용자에게 알림이 표시됩니다.
